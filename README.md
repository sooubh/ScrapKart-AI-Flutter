# ScrapKart AI ♻️

Welcome to **ScrapKart AI**, a comprehensive, AI-powered Flutter application designed for modern scrap management and recycling optimization. 

> **🎓 Academic Disclaimer**
> This project is developed as a **Final Year Project by Engineering students**. It demonstrates the integration of Artificial Intelligence with mobile application development using Flutter, alongside a robust backend infrastructure.

## 📱 About The Project

ScrapKart AI aims to revolutionize the way scrap is collected, categorized, and processed. By leveraging artificial intelligence and a sleek Flutter-based mobile interface, ScrapKart AI provides an intuitive platform for users to manage scrap efficiently.

### 🌟 Key Features
- **Flutter-Based Applications:** Includes the complete source code for the ScrapKart AI mobile apps, built with cross-platform compatibility in mind.
- **AI Integration:** Smart features for categorizing and valuing scrap items.
- **Backend Services:** Supported by a robust backend infrastructure (Node.js/Firebase) ensuring seamless data flow.
- **User-Friendly Interface:** A responsive and aesthetic UI developed for an optimal user experience.

## 🛠️ Tech Stack
- **Frontend:** Flutter & Dart
- **Backend:** Node.js, Express, Firebase (Firestore, Authentication)
- **AI/ML:** Integrated AI models for scrap analysis

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Node.js](https://nodejs.org/en/download/) (for backend services)
- [Git](https://git-scm.com/downloads)

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/karanxrathod/ScrapKart-AI-Flutter.git
   ```
2. **Navigate into the Flutter app directory:**
   ```sh
   cd "ScrapKart-AI-Flutter/scrapkart_ai"
   ```
3. **Install Flutter packages:**
   ```sh
   flutter pub get
   ```
4. **Run the App:**
   ```sh
   flutter run
   ```

*(Note: Make sure to set up your backend environment variables as per the instructions in the `backend/` directory).*

## 👨‍💻 Project Structure
- `/scrapkart_ai` - Main Flutter application.
- `/backend` - Node.js backend services and APIs.
- `/ScrapKart_Report` - Project documentation and reports.
